Windows小技巧 -- 目录内打开CMD的快捷方式

发表于 2016-11-10 | 更新于: 2018-06-13 | 分类于 Windows小技巧 | 阅读次数:
字数统计: 964 | 阅读时长 ≈ 4
本文记录Windows下目录内打开cmd命令行窗口的快捷方式。

在工作中常常会有需要在某个文件夹内使用cmd的情况，例如运行某脚本，下面演示几种方法。

以进入以下目录操作为例：

“姿势”一 ： 常用的cd命令

cd命令是我们平常使用比较多的方式：

  1. Win+R打开cmd窗口，默认显示如下（非管理员模式）；

  2. 此时需要依次键入命令e: cd github cd anyer cd Wechat-Weapp

或e: cd github\anyer\WeChat-WeApp\

或 cd E:\github\anyer\WeChat-WeApp\（没有进入指定目录时，再键入一个）

3.进行操作。

“姿势”二 ： 鼠标右键的快捷方式

显然经历了上面的多个命令，是不是感觉很忧桑，下面来个快捷点的方式：
1.Win+E资源管理器快速进入指定目录；

2.Shift+鼠标右键出现选项菜单

选择在此处打开命令窗口(W)项；

3.进行操作。

上述方式仅支持当前用户（非管理员）权限的cmd，当需要管理员权限时，可以尝试下面方式。

“姿势”三：资源管理器（2017-08-08 增）

在资源管理器的地址栏内输入cmd 或 powershell 回车后，依旧可以实现在当前目录快速打开命令行窗口:

  • 进入指定目录

  • 地址栏输入cmd 或 powershell

  • 回车确认

“姿势”四 : git命令行

此方式使用git的用户，安装git时确定安装git bash命令行，以确保可以在命令行里完成git操作~。windows下安装git教程自行百度了。安装好后，在需要使用cmd的目录中，使用git bash来替代使用。
1.资源管理器进入指定目录；
2.鼠标右键，菜单项中选择Git Bash Here项目；

3.进行操作；
